# AndroidStudio常用快捷键汇总


在这里记录平时在使用AndroidStudio过程中常用的快捷键，便于高开发效率，同时记录这篇博客便于后面查阅相关的快捷键。
由于Windows和Mac下的快捷键有所不同，这里将根据windows和mac两种环境下的快捷键分别记录起来。
目前只记录了windows下的快捷键，mac下快捷键篇，将在后面补上。

## windows下快捷键篇

### 视图切换快捷键

+ Alt+1 显示工程目录窗口（显示工程项目结构）
+ Alt+2 显示调试断点窗口&收藏窗口（查看已下的断点）
+ Alt+4 显示AppRun窗口（运行App）
+ Alt+6 显示debug和DDMS窗口（查看调试信息）
+ Alt+7 显示当前文件结构窗口（快速定位类文件或者布局文件结构）
+ Alt+9 显示VCS变更列表窗口（版本比较常用）

### 搜索快捷键

+ Ctrl+f 当前文件中搜索
+ Ctrl+O 显示当前文件中的方法名或者xml文件中的节点
+ Ctrl+G 选中一个方法或者变量，查找该变量或者方法的使用实例
+ 双击Shift 全局搜索任何内容

> 注意：AndroidStudio中所有的搜索都支持正则表达式和无序匹配关键字

### 编辑代码快捷键

+ Alt+Insert 在类定义之外，可以快速导入CopyRight,在类里可以快速生成Setter和Getter方法等。
+ Alt+Enter 快速提示，类似于Eclipse中的Alt+/ 。
+ Ctrl+Shift+f 快速格式化代码
+ Ctrl+Shift+O 快速导入依赖包

### 编译和运行快捷键

+ Ctrl+F9 编译整个项目工程
+ Ctrl+Shift+F9 编译工程中的某一个模块
+ Alt+Shift+X 运行工程中的某一个某块
+ Alt+Shift+D 调试工程
+ Alt+Shift+F9 调试项目
+ Alt+Shift+F10 运行项目配置

### 程序调试

+ F5 单步进入
+ F6 单步调试
+ F7 单步跳出
+ Ctrl+R 调试运行到光标处
+ Ctrl+Alt+F9 强制调试运行到光标处
+ F8 恢复程序正常运行
+ Ctrl+F2 终止调试
